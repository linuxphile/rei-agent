# /rei:offer

**Skills:** offer-generation, deal-analysis

**Workflow:**
1. Load or run deal analysis
2. Calculate MAO using all four methods
3. Assess market intelligence (DOM, price reductions, motivation)
4. Recommend offer strategy (aggressive/competitive/strong)
5. Draft LOI with recommended terms
6. Model 2-3 negotiation scenarios
7. Identify walk-away price

**Output:** MAO analysis + strategy recommendation + draft LOI.
