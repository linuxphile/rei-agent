# /rei:brrrr

**Skills:** deal-analysis, market-research, financing

**Workflow:**
1. Pull property data and ARV from sales comps
2. Estimate or input rehab budget
3. Model acquisition financing (hard money or cash)
4. Model post-rehab rents from rent comps
5. Model refinance at target LTV
6. Calculate cash left in deal, final CoC
7. Run BRRRR feasibility scorecard
8. Run sensitivity analysis on ARV and rehab costs

**Output:** Complete BRRRR analysis with feasibility score.
