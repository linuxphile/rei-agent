# /rei:compare

**Skills:** deal-analysis

**Workflow:**
1. Check if each address has an existing analysis in pipeline/
2. Run /rei:analyze for any missing
3. Generate side-by-side comparison table
4. Rank deals by deal score
5. Narrative comparison specific to investor goals

**Output:** Comparison table + ranking + narrative recommendation.
